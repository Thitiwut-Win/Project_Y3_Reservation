# Project_Y3_Reservation

# Event Booking Web App

A modern event reservation platform built with **Next.js**, **TypeScript**, and **Express.js**.
This web application allows users to browse available events, reserve tickets, and manage their bookings through a clean and responsive interface.

---

## Features

* **Authentication**

  * Secure registration, login, and logout using JWT.
  * Google OAuth integration for quick login.

* **Event Booking**

  * Browse events with live seat availability.
  * Reserve and cancel tickets with instant updates.

* **Modern Interface**

  * Responsive design with Tailwind CSS.
  * Optimized for desktop and mobile users.

---

## Tech Stack

**Frontend**

* Next.js 14 (App Router)
* TypeScript
* Tailwind CSS
* Axios

**Backend**

* Express.js (REST API)
* MongoDB + Mongoose
* JWT Authentication
* bcrypt for password encryption
